# Roadmap

## Phase 0: Validation

Duration: 1-2 weeks

Deliverables:

- confirm legal structure
- confirm bank account ownership
- confirm school cooperation model
- validate parent registration flow
- validate contribution model
- validate first task categories

## Phase 1: MVP Core

Duration: 6-10 weeks

Deliverables:

- auth integration
- onboarding
- tasks
- QR payments
- manual reconciliation
- feedback
- public reports
- admin panel
- Kubernetes deployment

## Phase 2: Engagement

Duration: 4-6 weeks

Deliverables:

- child ideas through parent submission
- voting
- events
- parent lectures calendar
- email notifications
- CSV bank import

## Phase 3: Automation

Duration: 6-8 weeks

Deliverables:

- automatic bank/payment provider reconciliation
- notification digests
- advanced reports
- PDF exports
- audit dashboard

## Phase 4: SaaS readiness

Duration: 2-3 months

Deliverables:

- multi-school tenancy
- billing for schools/spolky
- onboarding wizard for new schools
- configuration per school
- grant/sponsorship module
